# Changelog

## 1.0.0 Beta 1 - 2026-07-22

- Переработана единая тёмная тема, navigation shell и системные launch surfaces.
- Добавлены вкладка уведомлений, unread state, категории, read/clear actions и безопасные переходы.
- Добавлены настройки уведомлений, поддержка, правила и политика конфиденциальности.
- Добавлен public-safe offline cache для ленты, публикаций, комментариев и профилей.
- Улучшены альбомы, media playback, responsive layouts и accessibility states.
- Обновлены launcher icons, README и полный набор скриншотов интерфейса.

Beta 1 остаётся тестовой debug-сборкой и не предназначена для публикации в stores.

## 1.0.0 Alpha 1 - 2026-07-19

- Первая публичная Android alpha-сборка.
- Лента, публикации, профили, комментарии, реакции и upload/publish flow.
- Twitch-вход и восстановление eblo.id session.
